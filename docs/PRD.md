## 8. 디자인 가이드라인

### 8.1 레이아웃
- **Max-width:** 1200px
- **Padding:** 16px (모바일), 24px (태블릿+)
- **Gap:** 24px (섹션 간)
- **Header Height:** 70px

### 8.2 색상 사용
```css
Primary Action (CTA): #A78BFA (Soft Purple)
Secondary Action: #93C5FD (Sky Blue)
Hover State: Darker shade of primary
Success: #86EFAC (Mint Green)
Error: #F87171 (Soft Red)
Background: #FEFCE8 (Cream) / #1F2937 (Dark)
Text: #333333 (Light) / #F9FAFB (Dark)
```

### 8.3 컴포넌트
- **버튼:** 둥근 모서리 (12px), 패딩 12-16px
- **카드:** 둥근 모서리 (16px), 그림자 (0 4px 12px rgba)
- **입력 필드:** 둥근 모서리 (8px), 경계선 2px
- **하이라이트:** 부드러운 그라데이션 (Purple → Blue)

### 8.4 애니메이션
- **Transition Duration:** 300ms
- **Easing:** ease-in-out
- 호버 효과: 버튼 위로 이동 (2px), 그림자 강화
- 스크롤 진입: Fade-in + Slide-up
- 로드: Scale-in (프로필 이미지)
